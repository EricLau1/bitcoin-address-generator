# Gerador de Chaves e Endereços Bitcoin

```bash
    
    # generate private key
    npm run gen-pvt-key
    # output example:
    D6A16E871FD7087CC2C9396E7B252F4152932609E9018E87D4B1F76A98785743
    
    # generate wif
    npm run gen-wif D6A16E871FD7087CC2C9396E7B252F4152932609E9018E87D4B1F76A98785743
    # output example:
    5KSp2ogvJrtZaMafbMZGsRP9i19Vv5eTMCG9VrQcLZTz4z5j1Bi

    # generate public key
    npm run gen-pub-key D6A16E871FD7087CC2C9396E7B252F4152932609E9018E87D4B1F76A98785743
    # output example:
    04dadadd9db29bba908d73f0e3be1074a7f91b55cc378f3524dbb66a43b1313d62a950d32f8fe7b16821762fa84748b50173a92d4f03f954860e3a80e2a862a00c

    # generate bitcoin address
    npm run gen-btc-addr 04dadadd9db29bba908d73f0e3be1074a7f91b55cc378f3524dbb66a43b1313d62a950d32f8fe7b16821762fa84748b50173a92d4f03f954860e3a80e2a862a00c
    # output example:
    1DxaetPyWpfgeiPmqyL3pLERPJrMEJyJX8
```
